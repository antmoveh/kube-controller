#### kube-develop-tools是一个收集和学习kubernetes开发的项目，在该项目中会尽量使用各种方式来进行kubernetes的开发工作，在该项目中业务逻辑并非重点，笔者会设计非常简单的CRD资源，避免因为过多的业务逻辑导致理解困惑
#### 该项目类似一个工具书性质，在此查找实现并应用到自己项目中，因为并不会有大段详细的文章解释，但是会有必要的注释解释
